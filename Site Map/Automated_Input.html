Automated Input Process

Instructions

You will need the automated crusader extension which you can find HERE for chrome. This is the same extension that is used by Fantamondi to gather the information. You can see their comments Here if you like.

Once you have the extension you can go to your active COTLI game and click the extension in the top right and click the "Copy Data" button. Once you have done that, click this button and paste your data in.

Once you have input and parsed the data, you can check the results Here if you like.
